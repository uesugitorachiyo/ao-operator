# Query Patterns

Use focused searches and stop when the result is enough to guide the task.

## Manual Contract Lookup

```bash
qmd search "spec over prompt negative constraints verification loops" -c wiki
qmd search "provider runtime oauth cli schema boundary guards" -c wiki
```

## Manual Workflow Lookup

```bash
qmd search "subagent offload context window agent teams permissions" -c wiki
qmd search "worktrees code review tdd verification loops" -c wiki
```

## Manual Skill Lookup

```bash
qmd search "context window skills progressive disclosure verification loops" -c wiki
```

If exact searches return nothing, broaden once to a stable noun such as
`skills`, `verification`, or `context`, then read only the most relevant pages.
