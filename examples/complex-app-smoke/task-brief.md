# Complex App Smoke Task Brief

Build a small project tracker with users, projects, tasks, comments, and an
activity log.

Use AO Operator to:

- Classify the task.
- Shape it as greenfield.
- Produce scoped slices.
- Route implementation, review, integration, and evaluator closure through AO.

Constraints:

- Do not use network access.
- Do not read secrets.
- Do not create provider API-key dependencies.
- Keep all outputs as scaffold artifacts.
